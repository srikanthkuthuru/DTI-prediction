DrugKinet folder:

dti.csv: This file contains binary drug target interaction matrix. There are 725 rows corresponding drug compounds and 246 columns corresponding to kinases.

Ecfp4.csv: This file contains the Pubchem IDs, SMILES descriptors and circular fingerprint vectors (ECFPs) of all the 725 drug compounds

Kinases.csv: This file contains the names of all the 246 kinases mentioned before

PubChemIDs: This file contains the pubchem IDs of all the 725 compounds mentioned before

KinaseMatfiles: These .mat files contain the bioactivity data of each kinase with its interacting compounds directly collected from DrugKinet website
